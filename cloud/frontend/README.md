# MockingJay Dashboard

Web dashboard for viewing test results and metrics.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Open http://localhost:3000

## Features

- Real-time test results
- Pass rate metrics
- Average latency tracking
- Test history table

## API Connection

Dashboard connects to backend API at `http://localhost:8080` by default.

Make sure the backend is running before starting the dashboard.
